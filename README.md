## Введение
Это приложение для расчета ИМТ

## Примеры использования
Перед началом откроется таблица, в которой нужно ввести свой рост, вес, пол и возраст
Итоговый средний результат будет представлен в зависимости от возрастной каткгории пользователя
После просмотра результатов можно посмотреть таблицу лидеров и покажется рейтинг пользователя в ней

## Технические требования
Любой браузер (Yandex, Opera, Chrome)
